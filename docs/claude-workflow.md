- **Copywriting** — email drafts, social posts, session descriptions
- **Status reports** — give it the tracker, get a Slack-ready summary
- **Quality checks** — review copy against the style guide
- **Decision support** — vendor comparisons, tradeoff analysis
- **Vendor emails** — RFQ drafts, print specs, shipping timelines

## Best Practices

- **Start with context** — paste the relevant file URL before asking
- **Reference deliverables by ID** — "D1" or "the speaker posters"
- **Iterate** — push back, ask for alternatives, request revisions
- **Document AI usage** — note it in commits, it's part of the story

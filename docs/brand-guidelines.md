## Color Palette

### Base System (Established — Logo Presentation)

The foundation palette is set. All current production work uses these values.

| Name | Hex | Usage |
|------|-----|-------|
| Near-Black | `#111111` | Primary dark background — title and closing slides |
| Cream | `#EDEBE4` | Light background — concept and content slides |
| Amber | `#C8910A` | Accent — accent stripe in arc pattern, numbered list highlights |
| Off-White | `#F5F3EE` | Body text on dark backgrounds |
| Mid-Gray | `#888888` | Secondary text, captions |

### Expansion Direction (In Development — Full Visual System)

The base palette is intentionally restrained for the logo presentation phase. For the full event visual system — posters, badges, swag, environmental graphics — the color system expands to match the festival energy of the brief.

**Direction:** Keep near-black and cream as anchors. Layer in a vibrant, saturated secondary palette — warm magentas, electric cyans, deep purples, hot ambers. Think Coachella gradient energy, not corporate teal-and-orange. Colors should feel like they could exist on a concert poster or a festival wristband.

> Color expansion decisions will be locked in the Style Guide after logo direction is approved. Do not finalize secondary palette until the logo color system is confirmed.

## Typography

| Role | Font | Usage |
|------|------|-------|
| Display / Headlines | Century Gothic | All titles, slide headers, large type moments |
| Body / Supporting | IBM Plex Sans | Body copy, captions, rationale text, labels |

**Notes:**
- Century Gothic runs wide — size down 10-15% vs. Calibri equivalents to avoid wrapping
- IBM Plex Sans is a Google Font, free for all use, available at fonts.google.com
- Both fonts are already applied to the logo presentation template

## Illustration Style — Geometric Patterns

**Direction: Spotify Wrapped-style geometric pattern illustrations.**

The logo presentation establishes the arc motif (concentric quarter-circle stripes) as the primary geometric element. The full visual system expands this into a richer pattern language pulled from the Spotify Wrapped Figma reference.

**What we're using:**
- Bold geometric shapes: concentric circles, repeating arcs, dot grids, diagonal stripe fields
- Patterns used as texture, corner treatments, and full-bleed background elements
- High contrast: cream patterns on near-black, near-black patterns on cream
- Amber used sparingly as a single accent stripe within pattern sets — never as fill

**What we're NOT doing:**
- Illustrated characters or figures
- Gradient blobs or organic shapes
- Anything that reads as "tech startup" or "data visualization"

**Reference:** Spotify Wrapped Figma file — geometric pattern tiles and background treatments. See Visual References below.

## Creative Direction

- SXSW meets Coachella — festival energy, not tech conference
- Bold color, large-scale type, movement, texture
- Warm, inviting, slightly irreverent
- NOT cold, dark-mode, blue-glow futurism
- NOT corporate headshot grids or PowerPoint templates

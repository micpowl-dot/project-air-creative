## Context

The logo presentation was built and reviewed. Three logo concepts were explored and presented to the creative team. During this process, the visual direction for the full event system — beyond the logo — was defined. This decision log captures those directional choices so designers can execute without waiting for a live briefing.

---

## Decision

### 1. The Logo Presentation Establishes the Look and Feel

The current logo presentation (`assets/templates/logo-presentation.pptx`) is the visual foundation for everything downstream. The aesthetic: near-black and cream with a single amber accent, concentric arc patterns, bold sans-serif type, scribble texture overlay. This is the established baseline.

**What this means for your work:** When building the poster system, PPT template, or badge system — start from this look. Dark slides in near-black. Light slides in cream. Amber as the one accent hit. Do not introduce new colors until the expanded palette is approved.

---

### 2. The Color System Will Expand for the Full Event

The base palette is intentionally minimal during the logo phase. For the full event visual system, the palette expands into festival territory — vibrant secondary colors layered on top of the near-black/cream anchor.

**Direction:** Think Coachella gradient energy. Warm magentas, electric cyans, deep purples. NOT corporate blue, NOT muted sage, NOT tech startup teal-and-orange. Colors should feel like they belong on a concert poster or festival wristband.

**Timing:** Color expansion is locked after logo direction is approved. Do not design into the expanded palette yet. Build your systems with the base four colors and leave space for the palette to land.

---

### 3. Geometric Pattern Illustrations: Spotify Wrapped Reference

The primary illustration style for the event system is geometric patterns — the same visual language as Spotify Wrapped. Concentric circles, repeating arcs, dot grids, bold stripe fields. These run as background textures, corner treatments, and full-bleed elements.

**What this means for your work:**
- Posters: use pattern blocks as backgrounds or cropped corner elements. The pattern IS the design — it does not compete with the type, it supports it.
- Badges: use a simplified pattern as a lanyard or badge background element
- Swag: stickers and pins can use a single pattern motif as the primary graphic

**The Wrapped Figma is the reference:** https://www.figma.com/community/file/1580122793950964435

**What NOT to do:** gradient blobs, organic shapes, illustrated characters, anything that reads as "tech startup" or AI cliche (circuit boards, neural network nodes, glowing orbs).

---

### 4. Typography is Set

| Role | Font |
|------|------|
| Display / Headlines | Century Gothic |
| Body / Supporting | IBM Plex Sans |

Century Gothic runs wider than Calibri — size down 10-15% if text is wrapping unexpectedly. IBM Plex Sans is a free Google Font.

---

## Alternatives Considered

| Option | Why Not Chosen |
|--------|---------------|
| Full rebrand from green palette | Green felt too corporate; the Wrapped aesthetic better matches festival brief |
| Illustrated characters / figures | Too specific, ages fast, hard to scale across 17 ambassador posters |
| Single color palette (no expansion) | The brief calls for festival energy — the base palette is too restrained for posters and swag |

- Pattern illustration style (Wrapped geometric) is approved for use across all deliverables
- Color expansion happens after logo approval — no need to wait to start layout and type work
- Spotify Wrapped Figma is the primary reference — link it in your working files

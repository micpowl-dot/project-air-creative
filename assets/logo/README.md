# Project AIR — Logo Files

**Status: APPROVED** — Logo direction locked. These are the final production files.

---

## Which File Do I Need?

| Use Case | Folder | File to Grab |
|----------|--------|--------------|
| Print — posters, badges, swag | `PDF/` | Use PDF. Scalable, print-ready. |
| Digital — web, email, WorkVivo | `SVG/` | Use SVG. Scales to any size, no blur. |
| PowerPoint / Google Slides | `2x/` | Use PNG @2x. Crisp at presentation sizes. |
| Small screens / thumbnails | `0.5x/` | Use WEBP @0.5x. Optimized for small display. |
| Editing / extending the logo | `AI/` | Use AI (Illustrator source). Do not modify without checking with Michael. |

---

## Variants

Four versions. Use the one that fits your background.

| Variant | Filename Segment | When to Use |
|---------|-----------------|-------------|
| Full Color, Black "R" | `AIR_Color_blk-r` | On light/cream backgrounds |
| Full Color, White "R" | `AIR_Color_wht-r` | On dark/near-black backgrounds |
| All Black | `AIR_Black` | Single-color print, black on white |
| All White | `AIR_White` | Single-color print, white on dark |

---

## Folder Structure

```
assets/logo/
├── AI/       — Illustrator source files (.ai) — editing only
├── PDF/      — Print production (.pdf)
├── SVG/      — Digital/screen use (.svg)
├── 2x/       — High-res raster for slides/screen (.png)
└── 0.5x/     — Optimized small-display (.webp)
```

---

## Rules

- Do not stretch, skew, rotate, or recolor the logo
- Do not place the logo on a background that reduces contrast — use the correct variant
- Do not use the `AI/` files as final output — they are source files only
- Minimum size: do not use the logo smaller than 1 inch / 72px wide
- Clear space: maintain padding equal to the height of the "A" on all sides

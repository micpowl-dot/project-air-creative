## How This Repo Works

1. **Populate [Disciplines](docs/disciplines.md)** — add yourself, your tools, your availability
2. **Check the [Asset Tracker](assets/asset-tracker.md)** — see what needs doing
3. **Claim work** — put your name on a deliverable
4. **Create** — do the work, use AI tools, document what you used
5. **Review** — open a PR or flag it for team review
6. **Ship** — merge to main, update the tracker

### Branching

- `main` — approved, final content only
- `draft/*` — work in progress
- `review/*` — ready for feedback

### Commit Prefixes

- `[design]` — visual assets or design specs
- `[content]` — copy, messaging, scripts
- `[tracker]` — asset tracker or calendar updates
- `[docs]` — guides, disciplines, README
- `[swag]` — merchandise sourcing, specs
